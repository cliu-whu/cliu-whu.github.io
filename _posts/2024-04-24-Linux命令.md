# Ubuntu
查看软件有什么版本<br />`apt-cache policy`<br />指定版本安装<br />`apt-get install package=version`<br />查看包安装路径<br />`sudo dpkg -L`
# CentOS
查看已装软件<br />`rpm -qa | grep gcc`<br />查看安装路径<br />`rpm -ql gcc-4.8.5-44.el7.x86_64`
# MacOS
查看程序库链接<br />`otool -L`
